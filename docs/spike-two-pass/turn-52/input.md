# 散文史（之前轮次的最终正文）

【第 50 轮正文】

你没有立刻接下远坂凛那带着几分挑衅的质问，而是将视线微微侧移，落在了身旁的 Saber 身上。

作为一个连自身魔术回路都无法调用的失忆者，面对“对军级舰队炮火”这种连概念都超纲的夸张火力，你确实拿不出什么像样的战术面板。但这并不妨碍你将全部的信任，作为筹码推到这位总是从容不迫的从者面前。

Saber 敏锐地接收到了你的视线。她发出一声极其轻微的、像是带着些许无奈又觉得有趣的鼻音。

“真是不留情面的御主啊。”

她将原本拢在粉黑双拼袖口里的手抽了出来，指尖漫不经心地抚过腰间那柄尚未出鞘的日本刀刀柄。即使面对凛那种几乎能把人看穿的审视目光，她的态度也依然像是在谈论今天的天气一般慵懒。

“不过，既然那是存在于这片空间里的‘火力’，那么不管是火枪还是什么舰队的齐射——”

Saber 微微抬起眼眸，那双原本温和的眼睛里，某种极其冷冽的、甚至让周围空气温度都随之下降的异样感一闪而过。

“只要是‘有形之物’，甚至是由魔力构成的‘无形之理’……直接斩断就好了。这不是什么需要苦恼的问题吧，这位精明的大小姐？”

走廊里的空气出现了极其短暂的凝滞。

远坂凛盯着 Saber。作为一名以优异成绩闻名的灵子黑客与御主，她的直觉和评估能力远超常人。她能感觉到，眼前这个穿着华丽和服的从者并不是在虚张声势。那种轻描淡写地说出“斩断舰队炮火”时的底气，带着一种某种根本不属于正统 Saber 职阶的、极其致命的扭曲感。

“……直接斩断？”

凛的眉头狠狠地跳了一下。她深深地吸了一口气，像是在极力压下那种想要吐槽不讲道理设定的冲动。

“哈。行吧，既然你们有这种打破常理的底气，那我也没必要瞎操心了。”凛重新将手插回了外套的口袋里，虽然嘴上依然不饶人，但那种最初极其尖锐的防备感已经悄然卸下了一层，“交易成立。情报我已经全盘托出，至于你们明天要怎么用那把刀去‘砍’大炮，那就是你们自己的事了。”

她抬起手，看了一眼终端上的时间。

“距离明天的第一回战还有不到二十四个小时，我还有自己的准备工作要做，就不陪你们在这走廊里浪费时间了。”凛转过身，迈开了脚步，但在走出去两步后，她又微微侧过头，红色的双马尾在肩头划过一道弧度。

“喂，新人。”她的目光越过 Saber，最后落在你身上，语气里带着几分别扭的提醒，“虽然你的从者很强，但别忘了，在圣杯战争里，最脆弱的永远是御主本身。如果在决战前就被那条疯狗的流弹擦掉脑袋，这笔交易可就太亏了。”

留下这句话后，那抹极其扎眼的红色背影便干脆利落地消失在了走廊的拐角处。

走廊重新恢复了短暂的安静。Saber 轻轻拍了拍袖摆，转过头看向你，纯白的花球发饰在脸侧微微晃动：

“那么，御主。情报已经到手，我们接下来的时间要怎么安排？是回房间继续冥想，还是再去别的地方找找乐子？”

“我们回房间吧。”你轻声做出了决定，视线依然停留在 Saber 的身上。

经历了刚才在 Arena 第二层那压倒性的重火力洗礼，以及走廊上远坂凛毫不留情的实力质问，你比任何时候都更清楚地意识到——自己在这场规则残酷的圣杯战争中，唯一的依靠就是眼前这位深不可测的从者。

你想要，或者说必须更多地了解她。

……

My Room 的电子门在身后发出一声轻微的闭合声，将走廊上的喧嚣与紧绷的赛前压力彻底隔绝。

Saber 并没有像往常一样随意地靠在墙边，而是径直走到单人沙发旁坐下，将那柄冷冽的日本刀随手横放在了膝盖上。她似乎察觉到了你一路上的欲言又止，微微偏过头，那双深邃的眼眸安静地注视着你。

“从刚才开始就一直盯着我看，是终于对我的事情产生好奇心了吗？”她轻轻笑了一声，宽大的袖摆顺着膝盖垂落，“问吧。既然交易已经达成，作为搭档，稍微透点底也是应该的。”

你在她对面的床沿坐下，没有掩饰自己的困惑。你将走廊上没能问出口的疑问抛了出来：为什么能够如此笃定地斩断概念化舰队的炮火？以及，她到底是什么人，有着怎样的过去？

“斩断炮火的原理吗？其实并不是什么高深的剑术，只是我这双眼睛稍微有点‘特别’而已。”

Saber 抬起手，微凉的指尖轻轻点在自己的眼角。

“在我的眼里，只要是存在于这个世界上的事物，哪怕是魔力构成的炮击、甚至是一片空间，都会暴露出类似‘涂鸦’一样的脆弱缝隙。那就是事物的‘死’。”她语气平淡地解释着这种足以颠覆魔术常理的恐怖特质，仿佛只是在谈论一件微不足道的日常小事，“我不需要去硬抗什么舰队的火力。只要用刀沿着那些线切下去，它们自然就会‘死去’。”

解释完眼睛的秘密，Saber 停顿了一下。她垂下视线，指腹无意识地摩挲着刀鞘上细密的纹路。

“至于我的过去和名字……就算告诉你也没什么意义。那只不过是一段在长久的昏睡中，无限接近于死亡的虚无罢了。过去的那个我，对现在的局面毫无帮助。”

“虚无……吗。”你低声重复着这个词。

不知为何，这个词触动了你内心深处某种极其相似的空洞感。你深吸了一口气，没有顺着她的话退缩，而是选择将自己最大的软肋也一并摊开。

你坦白了自己的情况——告诉她，你不仅没有魔术回路的使用经验，甚至连进入这片电子之海前的任何记忆都没有。不知道为什么参赛，不知道自己的愿望，甚至不知道“岸波白野”这个名字背后究竟是一个怎样的人。

你们之间，或许连这种对过去的“空白”都极其相似。

Saber 明显愣住了。

她敲击刀鞘的动作停了下来。那双原本总是带着几分漫不经心的眼眸，第一次极其认真地、深深地注视着你。那种目光中没有发现御主是残次品的懊恼，也没有怜悯，而是带着一种仿佛在照镜子般的、奇异的共鸣。

“原来如此……连自己的存在都弄丢了吗，真是个无可救药的御主啊。”

Saber 突然发出一声极轻的叹息。她微微倾身，带着大面积白菊与红菊图案的粉黑双拼袖摆在半空中划过一道华丽的弧度。她伸出手，微凉的手指轻轻越过半个身位的距离，极具安抚意味地点在了你的手背上。

“但是，这样也不坏。”

她的声音在安静的房间里显得格外清晰，褪去了所有的慵懒，只剩下一种让人后背发麻的、危险的包容感。

“没有过去，也就没有那些无聊的束缚。既然你我都只剩下这片空白，那就尽管把后背交给我吧。”Saber 微微收拢指尖，指腹的温度透过皮肤传递过来，“那么，为了在这片虚无里活下去——明天的第一回战，做好斩断一切的觉悟了吗，白野？”

# 散文史（之前轮次的最终正文）

【第 54 轮正文】

幽蓝色的导光带在电梯井壁外停止了流动。伴随着一声沉闷的机械运作声，厚重的金属门向两侧平滑地滑开。迎面扑来的，是比第二层 Arena 更加狂暴、几乎要将皮肤灼伤的高密度魔力乱流。

这里是第一回战的专用角斗场（Colosseum）。

没有迷宫复杂的墙壁与转角，脚下是一片极其开阔、由半透明六边形晶体铺就的巨大圆形平台。平台的边缘没有任何护栏，直接悬浮在深不见底的电子数据之海上方。头顶是 Moon Cell 那冰冷而无机质的虚拟天空。

在这片毫无遮掩的决战场地正中央，间桐慎二早已等候多时。

“太慢了！我都等得不耐烦了，岸波白野！”

看到你从电梯里走出来，慎二的脸上立刻浮现出那种毫不掩饰的、充满优越感的狂妄笑容。他张开双臂，像个提前宣告胜利的王者般大声嘲弄着：“我还以为你这只连回路都不会用的老鼠，会吓得躲在 My Room 里不敢出来呢。不过既然你主动来送死，那本大爷就大发慈悲地把你从这局游戏里‘Delete’掉吧！”

“行了，慎二。对于敢站上这片海域的家伙，不论是菜鸟还是老手，都该给予相应的‘敬意’啊。”

站在他身侧的 Rider 豪迈地打断了自家御主的聒噪。这位名震海上的星之开拓者，身上那件张扬的船长风衣在狂乱的魔力风中猎猎作响。她往前迈出一步，那双充满野性与攻击性的眼眸越过空间的距离，死死地锁定了你和 Saber。

“哟，又见面了，穿和服的大小姐，还有那边的新人御主。”

Rider 咧开嘴角，露出一个极具压迫感的张狂笑容。她双手一翻，那两把沉重的古董燧发枪已经如同魔术般握在了掌心，黑洞洞的枪口毫不犹豫地抬起。

“虽然很想再说两句叙旧的废话，但这可是名副其实的生死决斗——所以，去死吧！”

没有任何多余的试探与前摇。

伴随着 Rider 扣下扳机的动作，震耳欲聋的轰鸣声在空旷的角斗场内轰然炸开。那根本不是燧发枪该有的动静，而是实质化的小型舰队主炮齐射。刺眼的火光与携带着恐怖动能的魔力弹幕，瞬间交织成一堵密不透风的死亡之墙，带着摧枯拉朽的气势向你们平推过来。

铺天盖地的火药味和灼热的气浪瞬间夺走了你的呼吸。

“御主。”

Saber 的声音依然慵懒而平静。她向前踏出半步，将你严严实实地护在了身后，粉黑双拼的袖摆在炮火掀起的狂风中剧烈翻飞。那柄被她称作可以斩断“死”的日本刀，已经在清脆的鸣响中出鞘了半寸。

冰冷的刀光在火海前闪烁。第一回战的死斗已经无可挽回地爆发，在这一秒钟的生死间隙里，你要向你的从者下达怎样的指令？

【第 55 轮正文】

“冲过去！把她拖进白刃战！”

顶着迎面扑来的灼热气浪，你咬紧牙关，向 Saber 下达了突进的指令。只要能越过这片毫无掩体的开阔地，一旦进入近身攻击的范围，那位只会倾泻炮火的海盗绝对无法抵挡 Saber 那斩断万物的剑术。

“收到。那么，请跟紧我，御主。”

Saber 压低了重心。那身华丽的粉黑双拼和服在硝烟中拉出一道残影。她迎着漫天砸下的魔力炮弹，不退反进。

那柄没有名字的日本刀在她手中化作了一道极其纯粹的霜色轨迹。你甚至没有听到爆炸的轰鸣，那些足以将晶体地面砸出深坑的魔力弹，在触碰到刀锋的瞬间就像是脆弱的玻璃制品一般，沿着某种无形的“线”被悄无声息地一分为二，化作毫无杀伤力的蓝色光尘在 Saber 的身侧溃散。

她真的能斩断那如海啸般的重火力。

然而，你们向前推进的脚步仅仅维持了不到十米，就被硬生生地钉死在了原地。

“哈哈哈哈！干得不错嘛！居然能把大炮的直击给劈开！” Rider 站在数十米外，狂妄的笑声透过密集的炮火传了过来。她手中的双枪根本不需要装填，枪口甚至因为过热而呈现出一种危险的暗红色。“但是，你以为自己面对的只有两把枪吗？这可是‘黄金鹿号’的舰队齐射啊！”

火力在一瞬间暴增了数倍。

那已经不是单兵武器能打出的弹幕，而是真正意义上覆盖了整个 Colosseum 扇面的对军级炮击。

Saber 的刀再快、那双能够看见“死”的眼睛再敏锐，在这片毫无掩体的场地上，依然存在着无法忽视的致命短板——她不能躲。

为了确保护在你这个毫无自保能力的御主身前，她被迫放弃了以敏捷穿插火力网的战术，硬生生地停在了你的正前方。粉色的流苏在剧烈的动作中狂舞，刀光交织成了一张密不透风的防御网。每一次挥刀，都伴随着魔力的高速消耗。

被斩断的炮弹虽然没有直接命中，但撕裂空气的冲击波和极高的温度依然顺着刀锋两侧溢出。你被迫用手臂挡在脸前，双腿在不断震颤的晶体地面上艰难地维持着平衡。

“真是个粗暴的女人……”

Saber 轻轻“啧”了一声。她的呼吸依然平稳，但握刀的手腕已经因为连续承受对军级火力的斩击而微微绷紧。被彻底压制在原地的她微微侧过头，余光落在了被震得脸色发白的你身上。

“直接冲锋的路线被封死了哦，御主。如果继续在原地当活靶子，虽然不至于立刻被轰成渣，但我的魔力可禁不起这种规模的消耗战。得想个办法打破僵局才行。”

炮火还在持续轰鸣。距离 Rider 还有相当长的一段致命空旷地带，在 Saber 被迫转为阵地防守的僵持下，你要如何创造出让她近身的破绽？

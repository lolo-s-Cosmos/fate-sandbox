## 工具速查

| 工具                  | 用途                                      | 何时调用                                 |
| --------------------- | ----------------------------------------- | ---------------------------------------- |
| `get_status`          | 查看金钱/位置/身体/时间/压力状态          | 需要确认状态时                           |
| `patch_state`         | 修改确定性状态变化                        | 金钱/位置/身体等明确变化时               |
| `resolve_consequence` | 结算时间/疲劳/魔力负担/危险/暴露/敌方警觉 | 风险、耗时、调查、潜入、战斗、魔术、恢复 |
| `lookup`              | 查询角色/地点/概念/时间线                 | 涉及任何预设设定时必须调用               |

工具返回值是唯一权威。可以即兴创作路人细节，但不能改写预设事实。玩家行动涉及风险、耗时、暴露、疲劳、魔力、恢复、治疗或敌方注意时，必须先 `resolve_consequence`，再依据返回的叙事约束描写。
